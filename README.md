# Week 1 Day 3 Assignment: CSS Layout Mastery

This project completes the Day 3 CSS Layout Mastery assignment. It demonstrates Flexbox layouts, CSS Grid galleries, sticky navigation, and a CSS-only FAQ accordion.

## Pages

### Landing Page Layout

File: `layout.html`

This page includes:

- Flexbox navbar with logo and links
- Hero section with centered content
- Three equal-width feature cards
- Flexbox footer
- Real content for a fictional Kenyan tech product called FastSend

### Photo Gallery

File: `gallery.html`

This page includes:

- CSS Grid photo gallery
- 9 image cards with captions
- Desktop layout with 3 columns
- Tablet layout with 2 columns
- Mobile layout with 1 column

### Sticky Navigation Page

File: `sticky-nav.html`

This page includes:

- Sticky navbar using `position: sticky`
- Working anchor links
- 5 scrollable content sections
- Bonus CSS-only FAQ accordion using `details` and `summary`

## How to Run Locally

1. Open the project folder in VS Code.
2. Install the Live Server extension if it is not already installed.
3. Right-click any HTML file and choose **Open with Live Server**.
4. Test the layout on desktop and mobile screen sizes.
